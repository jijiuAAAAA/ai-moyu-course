# AI摸鱼王 - Day 1 课程
## Excel 自动化基础

### 1. AI辅助公式生成

**适用场景：** 不知道用什么公式、公式写错了

**工具：** ChatGPT / Claude

**提示词模板：**
```
我的Excel表格A列是商品名称，B列是销售额，C列是销售数量。
请帮我写一个公式，计算每个商品的平均单价。
```

**效果：** AI直接给出公式 =B2/C2

---

### 2. 一键生成图表

**工具：** Excel内置AI / ChatGPT

**步骤：**
1. 选中数据区域
2. 点击"插入"→"智能图表"
3. 选择合适的图表类型

**技巧：** 告诉AI你想要什么类型的图表，它会推荐

---

### 3. 数据自动清洗

**常见问题：**
- 空格多余
- 大小写不一致
- 重复数据

**AI处理方法：**
```
请帮我写一个Excel公式，去除A列单元格中的多余空格，并把所有文本转为大写。
```

**公式：** =TRIM(UPPER(A2))
